Kontakta Toastmasters via mejl

Här kan ni skicka mejl som går till [Namn och Namn(Smeknamn)]

Om ni ska göra spex eller tal, tänk på att skriva hur lång tid ni behöver för att underlätta planeringen för våra kära toastmasters.

Om ni inte får svar så kanske erat mejl inte har skickat, kolla i eran epost först.

Ni är alltid välkomna att ringa till Namn 1 eller Namn 2 om ni inte får något svar eller inte kan skicka mejl så att vi kan kontaka dem åt er.
